# Recipe App Frontend

A React + Vite frontend for a Recipe Management System that allows users to browse recipes, leave reviews, rate recipes, and manage their favorites.

## Features

- Browse and view recipes
- Rate recipes with star ratings
- Write and submit reviews
- Toggle recipes as favorites

## Tech Stack

- React + Vite
- Vitest for unit testing

## Getting Started

```bash
npm install
npm run dev
```
